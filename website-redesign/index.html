Heuristic Usability Evaluation

A detailed usability audit of the original financial portal conducted using Jakob Nielsen's 10 Usability Heuristics, alongside the redesign outcomes.

Heuristic 8

Aesthetic & Minimalist Design

The interface should not contain information that is irrelevant or rarely needed. Every extra unit of information competes with relevant units and diminishes their relative visibility.

Original Design

  • Extreme visual noise (blinking warnings, aggressive ads, complex cryptographic status text).
  • Redundant labels and dense borders creating visual cognitive fatigue.

Redesigned Solution

  • Clean dark-mode palette, subtle grid lines, high-contrast borders, and spacious layout.
  • Focused metrics, interactive cards, and hidden utility icons to minimize distractions.
Heuristic 5

Error Prevention

Even better than good error messages is a careful design which prevents a problem from occurring in the first place. Either eliminate error-prone conditions or present a confirmation option before committing.

Original Design

  • Destructive transfer action starts immediately via a massive red button with no fields check.
  • Input fields allow users to type alphabetic characters into the dollar value field, causing raw database errors.

Redesigned Solution

  • Pre-validated numeric ranges tied to available reserves, autocomplete dropdowns, and an explicit transaction clearance toggle.
  • Prompt modal reviews and dynamic warning banners block invalid operations prior to network routing.
Heuristic 1

Visibility of System Status

The system should always keep users informed about what is going on, through appropriate feedback within a reasonable time.

Original Design

  • Initiating transfers caused screen freezes without displaying progress meters.
  • Alert states are consistently persistent, offering no indication when issues have resolved.

Redesigned Solution

  • Interactive status indicators: spinner loaders upon click, dynamic balance deduct animations, and a rich confirmation success view.
  • Pulse indicators and alert badges dynamically flashing to display connection states.
Heuristic 7

Flexibility & Efficiency of Use

Accelerators — unseen by the novice user — may often speed up the interaction for the expert user such that the system can cater to both inexperienced and experienced users.

Original Design

  • Users have to type details line-by-line; no quick filters for dates or quick search.
  • No keyboard accessibility; tiny buttons targetable only via careful mouse clicks.

Redesigned Solution

  • "MAX" button instantly fills balance calculations; search filters update transaction tables as-you-type.
  • Responsive spacing, focus states, and tab-key navigations for desktop shortcut operation.
